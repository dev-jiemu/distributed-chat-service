Distributed Chat Service

RabbitMQ와 Redis를 활용한 확장 가능한 채팅 서비스

STOMP Chat Client

Quick Start

  1. 위의 "STOMP Chat Client" 클릭
  2. User ID 입력 (예: user1, user2)
  3. Connect 버튼 클릭
  4. 받는 사람 ID 입력하고 메시지 전송
  5. 여러 브라우저/탭에서 다른 User ID로 접속하여 테스트

기술 스택

  • Spring Boot 3.3.0 + WebSocket (STOMP)
  • RabbitMQ (Topic Exchange)
  • Redis (연결 정보 캐싱)
  • Docker Compose
  • SockJS + STOMP.js

주요 기능

  • ✅ 1:1 실시간 채팅
  • ✅ 분산 서버 간 메시지 라우팅
  • ✅ 접속/종료 알림
  • ✅ 타이핑 표시
  • ✅ 읽음 확인
  • ✅ 발신자 에코백
  • ⏳ 채팅방 기능 (구현 예정)
  • ⏳ JWT 인증 (구현 예정)
